# UI 验收清单（持续更新）

本清单用于人工验收与 CI 可视化检查的对照表。

- 字体与中文显示
  - 主题 `theme.json` 中 `label/button/window` 使用 `assets/fonts/汇文明朝体.otf`
  - 所有中文文案无“口口”/乱码/空白

- 设置窗口（Settings）
  - “音频/显示/语言”页签可见、可点击
  - 音频页：主音量/音乐/SFX 数值与滑杆交互实时同步；“静音”状态切换正确；“测试音效”可听见
  - 显示页：分辨率下拉含常用项与当前桌面；“应用”后出现 10 秒确认提示；“确认”保存、“撤回/超时”回退
  - 语言页：当前仅 `zh-CN` 占位；关闭时写入 settings.json

- UI 尺寸与警告（UILabel/UIButton/UIWindow）
  - 运行时无 `UserWarning: Label Rect is too small for text` 等尺寸警告
  - 若出现警告：调整相应控件 `relative_rect` 宽高，或启用自动换行/缩短文本，直至警告消失

- 交互阻塞与输入锁
  - 打开“学术八卦”/“来自任务线的小道消息”对话框时，背景 UI 不可点击
  - 仅弹窗关闭按钮（或 ESC/标题栏X）可关闭并恢复输入

- 战斗 UI
  - 能量展示为一排彩色方块；总数无限制
  - 回合开始有色能量清零、无色补满；每个已激活理想+1对应色
  - 仅无色也可支付任一颜色卡牌（优先同色再无色）

> 注意：本清单为人工/可视化验收指南，不替代自动化测试；对应关键交互需有单元/E2E 测试覆盖。***
