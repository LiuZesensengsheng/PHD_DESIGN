## 五色高层设计思想 v1（轴心与性格草案）

> 本文档汇总当前对白 / 红 / 蓝 / 绿 / 黑五色的整体设计思路，作为各色 `*_IDEAL_CARD_DESIGN` 的上位纲领。  
> 重点记录：每个颜色的核心轴、性格定位、与 buff / 状态 / 资源的关系，为后续扩展与裁剪提供参照。

---

### 0. 通用卡牌设计铁律（机制复杂度约束）

- **单张卡牌最多只挂 1–2 个机制轴**  
  - 设计每张牌时，必须先在“机制轴”中标注：`机制轴：A（必选），B（可选）`；  
  - 若出现第 3 个独立机制轴（例如：同时吃创新 + 非红 + 出牌数 + 燃尽），则视为超标设计，需要拆分或删减。

- **复杂度来自“机制之间的递增组合”，而不是单牌内部的 if/else**  
  - 深度主要由：  
    - 相同机制轴下不同牌的叠加、清算、转换（如：创新次数→伤害/防御/能量）；  
    - 不同机制轴之间的组合（如：创新 + 非红、解析 + 成长）。  
  - 单张牌内部若出现过多条件分支（多个“若/如果/当……且……”），则优先拆成两张更专一的牌。

- **坏味道判定**  
  - 当一张牌的文本需要解释超过 2 个“前提条件”才能看懂其主效用时，即可判定为“坏味道设计”，应回炉重构。  
  - 目标是：**玩家在 1–2 秒内就能读懂一张牌的主作用，把真正的复杂度留给牌之间的搭配。**

### 1. 白色：排序与控场（指针排序轴）

- **核心性格**  
  - 稳健、理性、偏“结构控”：  
    - 擅长**重排牌序 / 重排敌人行动顺序**；  
    - 倾向通过“视角、结构化”来赢牌，而不是纯数值爆发。

- **主轴**  
  - 指针排序轴（详见 `WHITE_IDEAL_CARD_DESIGN_V3`）：  
    - 通过“指针 / Pivot / ViewOrder”等机制，操纵牌序与行动顺序；  
    - 以控场、防御和中期算计为主。

- **与 buff / 状态关系**  
  - 白色不以 buff 堆叠为主，而是以**结构与顺序的优势**替代传统 buff：  
    - 例如：固定抽牌顺序、锁定敌人行动节奏等等。

---

### 2. 红色：创新 + 交叉学科 + 随机（影响轴）

- **核心性格**  
  - 冲动、连打、短期爆发、敢赌：  
    - 擅长“**抽上来就打光**”的高节奏输出；  
    - 喜欢拿来主义：大量使用别色牌与卡组外牌。

- **主轴一：创新轴（本回合新抽牌）**  
  - 关键词：**创新牌 = 本回合新抽上手的牌**。  
  - 核心玩法：  
    - 创新共振能力牌：当你打出创新牌时抽牌；  
    - 创新次数作为派生计数（连续创新次数 → 爆发伤害 / 防御 / 自疗）；  
    - 一系列只吃“创新”或“是否为创新牌”的攻防与资源牌。  
  - 风格：**即时兑现** + 连打爽感。

- **主轴二：交叉学科（非红牌 + 卡组外牌）**  
  - 非红轴：  
    - 以“本回合打出过非红牌”为条件，放大红色攻击 / 防御 / 吸血 / 降费。  
  - 卡组外轴：  
    - 利用战斗中临时获得的牌（小结、小刀、他色攻击等）作为“外来理论”；  
    - 通过能力牌与技能牌放大“卡组外牌”的伤害、资源与连打潜力。  
  - 风格：**拿来主义** + 多色协同。

- **主轴三：随机轴（高波动导入/导出）**  
  - 通过“随机抽牌 / 随机发现 / 随机目标 / 随机奖励”制造波动：  
    - 小结模板 + 每回合/一次性生成小结；  
    - 随机三选一攻击牌；  
    - 随机目标爆发的群体伤害技能等。  
  - 随机性不是独立资源，而是**创新与交叉学科的高波动变体**。

- **与 buff / 状态关系**  
  - 红色不主打复杂 buff，而是通过：  
    - “创新次数”“本回出牌数”“非红计数”“卡组外计数”等**短期计数器**来获得爆发。

### 3. 蓝色：解析（毒） + 力量战式成长 + 安全型 buff

- **核心性格**  
  - 冷静、长期主义、偏“研究和优化”：  
    - 前期偏弱，后期极强；  
    - 通过“毒 + 属性成长 + 精细 buff”滚雪球。

- **主轴一：解析轴（蓝色版毒）**  
  - 「解析」 = 持续伤害状态，可以照搬“毒”的基本体验：  
    - 敌人每回合受到等同于解析值的伤害（衰减与否可在蓝色文档中细化）；  
    - 大量牌负责叠解析、放大解析、用解析值清算大爆发。  
  - 解析体现蓝色的**信息优势与长期压制**。

- **主轴二：力量战式自我成长**  
  - 引入类似“力量/敏捷”的蓝色专属属性层数，例如：  
    - 研究深度：每层让解析牌多施加 1 点解析；  
    - 思维清晰：每层让技能牌提供更多防御/抽牌。  
  - 成长可以在**战斗内**或**战斗外**发生：  
    - 战斗内：每回合满足某条件（施加足够解析、打出足够技能）就涨层数；  
    - 战斗外：长期研究进度条，解锁更高上限。  
  - 风格：典型的“**力量战蓝色版本**”：  
    - 开局虚，叠层数与解析；  
    - 中后期靠叠满的层数碾压战场。

- **与 buff / 状态关系**  
  - 蓝色擅长给**己方与敌方上“安全、可控的 buff/状态”**：  
    - 己方：提高解析效率、提升抽牌质量/能量节奏、增强某类牌的效果；  
    - 敌方：施加易解析、下回合伤害翻倍但解析也翻倍等“可被利用的条件”。  
  - 关键词：**控制、调参、实验设计**，风险总体可控，不以自残为主。

### 4. 绿色：卡牌与防御的成长 + 斩杀

- **核心性格**  
  - 稳健、耐心、偏“养成与防守反击”：  
    - 通过卡牌升级、防御叠加、条件斩杀来赢牌；  
    - 很适合喜欢慢慢养成、一击终结的玩家。

- **主轴一：卡牌无限升级机制**  
  - 战斗内升级：  
    - 某些绿牌每打出一次，本战中伤害/防御/效果+1；  
    - 或通过特定技能让手牌/牌堆中的某牌在本战内升级。  
  - 战斗外升级：  
    - 绿色可以包含“升级器”牌，用来强化牌堆中的其他牌：  
      - 击杀敌人后升级；  
      - 或在特定条件下（例如低血、防守时）升级。  
  - 风格：强调**“牌本身的成长”**，与蓝色主打“人/属性成长”形成对照。

- **主轴二：防御 + 斩杀**  
  - 防御：  
    - 绿色偏向高值、粗犷、可叠加的护甲与回复；  
    - 与白色的“精细控场”区别开来，更像“筑墙”。  
  - 斩杀：  
    - 据敌人当前生命比例 / 已承受伤害 / 已叠状态等触发强力终结技；  
    - 与卡牌升级互动：升级后的攻击牌在斩杀条件下成为极高收益的终结技。  
  - 绿色整体是“**活得越久越不怕**”的那一色。

- **与 buff / 状态关系**  
  - 绿色侧重给**自己上成长与防御型 buff**：  
    - 护甲持续增长、回复增强、卡牌数值逐渐变大；  
    - 几乎不自残，也不鼓励高波动博弈。

---

### 5. 黑色：Buff 堆叠 + 诅咒利用 + 自残换爆发

- **核心性格**  
  - 极端、冒险、双刃剑：  
    - **最有意思，也最容易死**的卡组；  
    - 用血量、手牌、健康牌堆作为赌注换取爆发和扭曲性资源。

- **主轴一：越弱越强的 buff 体系**  
  - 给自己上的是各种“条件化的正面状态”：  
    - 生命越低，攻击越高；  
    - 手牌越少，下回合抽牌越多；  
    - 身上负面状态越多，输出/资源越爆表。  
  - 这些 buff 往往要求玩家**把自己推向危险边缘**。

- **主轴二：诅咒与自残利用**  
  - 诅咒不是单纯的坏牌，而是黑色的“特殊资源”：  
    - 手牌/牌堆中的诅咒数量作为爆发系数；  
    - 可以通过特定牌消耗诅咒来换一次超大伤害或资源爆发。  
  - 自残形式：  
    - 降低自身最大生命 / 当前生命；  
    - 减少手牌上限 / 抽牌数；  
    - 主动给自己挂负面状态，再由其他牌转化为收益。

- **与 buff / 状态关系**  
  - 黑色擅长给**自己和敌人同时上 buff/状态**，制造高风险局面：  
    - 给敌人上强 buff，再用“按敌人 buff 数量清算”的牌收割；  
    - 同时身上也挂大量 buff 与 debuff，形成复杂博弈。  
  - 与红色短期透支的区别：  
    - 红：更多是“一两回合的节奏透支”；  
    - 黑：更接近“长期契约”，越打后果越严重，但也越可能一拳带走战斗。

---

### 6. 五色 buff 分工小结

- **白色**：  
  - 偏向“结构与顺序控制”，不是典型 buff 色。
- **红色**：  
  - 用短期计数器（创新次数 / 出牌数 / 非红数 / 卡组外数）代替传统 buff。
- **蓝色**：  
  - 擅长给己方与敌方上**安全、可控的 buff/状态**，强调长期效率与滚雪球。
- **绿色**：  
  - 擅长给自己上**成长 + 防御型 buff**，牌和护甲越磨越厚。
- **黑色**：  
  - 擅长给**自己和敌人同时上 buff/诅咒**，是“高风险高收益”的极端 buff 色。
