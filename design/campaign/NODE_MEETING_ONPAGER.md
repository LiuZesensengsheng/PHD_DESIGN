### 1. 概述与设计意图 (Elevator Pitch)

**一句话总结：组会是游戏中的“篝火”与“商店”，是玩家在紧张的甘特图推进中获得喘息、进行战略构筑、并与导师（最终Boss）互动的核心节点。**

“组会节点”在主题上是博士生每周与导师的例行会议，在功能上则是Roguelike游戏中的核心休整点。它不是甘特图轨道上的一个“块”，而是一个强制触发的、不消耗回合的特殊阶段。它旨在为玩家提供一个安全的环境，将战斗中获得的资源（灵感）转化为长期战力，并构成游戏的核心决策循环。

### 2. 出现时机与规则 (Trigger & Rules)

- **触发周期**: 在甘特图时间轴上，**每50天 (5个回合)** 固定触发一次。
- **初始触发**: 在入学的**第0天**会额外触发一次，作为**新手引导、初始构筑、并发布第一个主线任务**。
- **总计次数**: **8次**（第0天，5回合，10回合，15回合，20回合，25回合，30回合，35回合）。
- **时间消耗**: 进入组会节点**不消耗**任何游戏天数。
- **强制性**: 这是一个强制触发的事件，玩家不能跳过。
- **表现形式**: 在甘特图UI上，它会以一个特殊的图标（如一个“会”字或一个日历标记）出现在对应时间点上，作为预告。
-- **任务线出现约束**: 结合当前状态新增任务线；当已有任务线达到**4条**上限后，本次及后续组会**不再新增**任务线（仍可提供其它对话/奖励选项）。

#### 2.1. 第0天初始组会特殊规则

第一次组会是玩家旅程的起点，除了提供常规的构筑功能外，还承担着关键的新手引导和目标设定作用。

- **新手引导 (Tutorial)**: 导师的对话将作为游戏的核心教程，以自然的方式向玩家介绍**灵感点数**的用途、**卡组构筑**的重要性（特别是移除卡牌）、以及**组会**的游戏节奏。（这部分先不做）

- **发布初始任务 (Quest)**: 导师会发布一个贯穿前期游戏的主线任务，为玩家提供清晰的短期目标。
    - **任务示例**: “在下次组会前，完成对‘横向课题’的初步探索（完成3个横向课题的甘特块）”。
    - **奖励示例**: 完成后，在下次组会时获得大量灵感点数或一张强力稀有卡牌。

### 3. 核心功能与流程 (Detailed Flow)

> **核心设计**: “组会”专注于**战略成长**，而**战术生存**(回血)则交给玩家主动选择的“摸鱼”行动。

**交互流程总览:** 

这部分我想简洁一点，能变成就是一个固定的商店页面吗，里面大概分3个区域，一个区域是导师的部分，加入 删除 和 升级的按钮
第二个区域 要 是师兄师姐的界面， 提供购买卡牌的功能。 第三个区域 先不定内容，但是是购买的特质的部分。


玩家进入组会场景后，界面中央是导师，周围环绕着 3 个核心交互选项。玩家可以自由点击执行，直到选择“结束组会”离开。

**1. 方向指导 (免费卡牌升级 - 每轮一次)**
- **功能**: 这是每次组会的**固定福利**。导师会提供一次免费的“方向指导”。
- **流程**: 玩家点击此选项，会弹出自己的牌库，选择一张卡牌进行**免费升级**。每次组会此选项只能使用一次，使用后按钮变为灰色。
- **目标**: 提供保底成长，降低玩家挫败感，并让后续消费决策更有策略性。

**2. 学术交流 (商店)**
- **功能**: 消耗“灵感点数”购买卡牌或特质。
- **商品池 (EA版本)**: 每次固定刷新 **5** 张卡牌：
    - **2张** 常见品质卡牌
    - **3张** 稀有品质卡牌
- **刷新策略（MVP）**: 本版本不提供刷新；**每次进入组会时生成一次固定候选**，页面内操作不触发刷新。
- **价格（占位）**: 卡牌/特质统一价格为 **80 灵感**（占位，可后续细化为分档/稀有度系数）。
- **目标**: 玩家拓宽构筑思路、获取新卡牌的主要途径。

> **未来设计接口**: 在后续版本中，可以将一个“稀有卡牌”槽位升级为出售“特质传承挑战”的特殊商品，以增加长期成长维度。

**3. 进度汇报 (付费卡牌升级)**
- **功能**: 消耗“灵感点数”来升级卡牌，可多次使用。
- **流程**: 玩家点击 -> 弹出牌库 -> 选择卡牌 -> 确认弹窗显示消耗与效果 -> 确认后执行升级。
- **消耗（占位）**: **80 灵感/次**（后续可加入随次数递增与稀有度系数）。
- **目标**: 强化现有核心卡牌，提升卡组上限。

**4. 优化论文 (移除卡牌)**
- **功能**: 消耗“灵感点数”来精简卡组。
- **流程**: 玩家点击 -> 弹出牌库 -> 选择卡牌 -> 确认弹窗显示消耗 -> 确认后执行移除。
- **消耗（占位）**: **80 灵感/次**（动态调价规则留待后续版本）。
- **目标**: 提升卡组稳定性，是高级玩家的核心追求。

**5. 事件（组会对话与选项，MVP）**
- 公共约束：本次组会仅触发一条事件；无回合消耗；奖励与状态即时生效。
- 统一字段（工具性定义）：`id`/`title`/`who`(导师/师兄/自语)/`trigger`/`choices[]`(text,effects)/`no_turn_cost`=true
- 常见效应：`add_track(...)`、`grant_inspiration(n)`、`grant_card(pool,count)`、`grant_trait(pool)`、`set_flag(key,val)`、`hint(text)`

- 模板A｜导师检查工作量（关联最近进度）
  - 触发：进入组会；近5回合完成≥1个任务块→A分支，否则B分支
  - A分支 选项：
    - “继续保持” → `grant_inspiration(20)`
    - “加码挑战” → `set_flag(hard_mode_next=1)`；下次组会商店“稀有位+1”（不改价）
  - B分支 选项：
    - “临时加速” → 目标任务线“完成度+1格”（不超上限）
    - “承诺改进” → `set_flag(must_finish_in_5_turns=1)`；若5回合内完成≥1块，则在下次组会结算 `grant_inspiration(40)`

- 模板B｜新增任务线（≤4条时才会出现）
  - 触发：当前任务线数 < 4；触发者权重：导师50%｜师兄30%｜自发20%
  - 选项（三选一，占位蓝图）：
    - “导师指派横向课题” → `add_track(type='横向', total=4, theme='应用')`；完成首块时 `grant_inspiration(30)`
    - “师兄拉活儿” → `add_track(type='会议', total=3, theme='汇报')`；完成全部后赠“特性三选一”
    - “自主立题” → `add_track(type='论文', total=6, theme='基础')`；难度+1，完成结尾时“稀有卡+1”
  - 限制：若已达4条，则本类事件退化为“风味/小奖励”（如 `grant_inspiration(20)` 或 `hint(...)`），不新增任务线

- 模板C｜风味事件（轻叙事 + 轻收益）
  - “咖啡与灵感” → 选“再来一杯”：50% `grant_inspiration(15)` ｜ 50% `hint("失眠一夜（无数值效果）")`
  - “组会幻觉” → 选“精神胜利”：`hint("你突然觉得PPT自己动了起来")`
  - “PPT模板争论” → 选“套模板” `grant_inspiration(10)` ｜ 选“纯手工” → “下一次购卡 -10（一次性折扣）”

### 4. 期望体验 (Desired Experience)

- **重大节点**: 每一次组会都是一个值得期待的“大事件”，是玩家构筑思路成型、战力大幅提升的关键点。
- **战略决策**: “灵感点数”是有限的，玩家必须在“买新牌（广度）”、“升级旧牌（深度）”、“移除废牌（纯度）”这三个核心构筑维度上做出取舍。
- **叙事窗口**: 通过导师的几句对话， 逐步地揭示世界观、主角的心路历程。导师的反馈会与玩家的“任务完成度”等状态挂钩。**PPT屏幕上的内容也会随着游戏进程（如第1、2、3次组会）发生变化**，为最终的Boss战埋下伏笔。

### 5. 数值占位（MVP）

- **初始灵感**: **50**
- **价格基准**: **80 灵感**（付费升级、删除卡牌、购买卡牌与特质）
- 后续将根据产出/消耗闭环与难度曲线进行统一平衡与细化。
