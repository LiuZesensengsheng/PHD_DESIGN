## 助教线·手机消息段子库（V1 草案，仅内容台账）

本文件用于**先记录内容**，不涉及实现与数值。

目标：把助教线“段子消息”整理成统一的**三段式**：
- **标题**（通知标题）
- **正文**（消息内容）
- **吐槽按钮**（替代弹窗关闭 `X` 的按钮文案；点击即关闭/返回）

未来可能加入**表情包/图片**，因此本台账预留 `media` 接口字段。

---

### 1) 未来数据结构（建议，供实现时对齐）

最小字段（兼容现有 `data/flavor/line_bubbles.json`）：
- `id`: string
- `title`: string
- `text`: string

扩展字段（本台账新增）：
- `button_text`: string（吐槽按钮文案；为空则用默认“行吧”/“我懂了”）
- `media`（可选）：用于表情包/图片
  - `kind`: `"sticker"` | `"image"`（后续可扩展）
  - `asset_id`: string（资源ID；由 UI/资源系统解析）
  - `alt`: string（无图时的替代文本/可访问性描述）

示例（仅示意）：

```json
{
  "id": "ta_msg_001",
  "title": "作业夹带私货",
  "text": "学生交的实验报告里夹着奶茶店积分卡，背面写着“求捞”。",
  "button_text": "你这属于学术贿赂（未遂）。",
  "media": { "kind": "sticker", "asset_id": "sticker_ta_tea_card", "alt": "奶茶积分卡表情" }
}
```
