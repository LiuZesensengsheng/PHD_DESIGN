## 学术八卦·手机消息段子库（V1 草案，仅内容台账）

本文件用于**先记录学术八卦内容**，不涉及实现与数值。

目标：把“学术八卦频道”整理成统一的**三段式**：
- **标题**（通知标题 / 频道标题）
- **正文**（消息内容）
- **吐槽按钮**（替代弹窗关闭 `X` 的按钮文案；点击即关闭/返回）

未来可能加入**表情包/图片**，因此本台账预留 `media` 接口字段。

---

### 1) 未来数据结构（建议，供实现时对齐）

最小字段（兼容现有 `data/flavor/line_bubbles.json` 的条目形态）：
- `id`: string
- `title`: string
- `text`: string

扩展字段（本台账新增）：
- `button_text`: string（吐槽按钮文案；为空则用默认“离谱”/“我不懂但我大受震撼”）
- `media`（可选）：用于表情包/图片
  - `kind`: `"sticker"` | `"image"`
  - `asset_id`: string
  - `alt`: string

---
