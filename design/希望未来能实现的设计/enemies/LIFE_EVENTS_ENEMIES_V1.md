敌人设计 · 生活事件原型 V1
================================

> 目标：只记录“生活梗 + 大致战斗感觉”，暂不设计数值/技能细节，后期从中挑选代表敌人进入正式敌人池。

1. 昨天的 DDL（RED）
--------------------
- 场景：深夜赶稿 / 项目申报。
- 现实梗：明明今天才是截止日，但它总是以“昨天”的姿态压在你身上。
- 战斗感觉（粗略）：时间炸弹型——倒计时几回合后打出超高爆发；你每拖一回合，它的强度就叠高。

2. 食堂没变过的菜（GREEN/WHITE）
-------------------------------
- 场景：中午去食堂打饭。
- 现实梗：永远是那三样菜，你吃不胖，只会吃到精神耗尽。
- 战斗感觉（粗略）：慢性消耗型——每回合给你一点小压力/小 debuff，比如减抽牌、减灵感，拖得越久越难受。

3. 早上的闹钟（RED/WHITE）
------------------------
- 场景：早八课 / 早上实验排班。
- 现实梗：每天都响，但你从来没在第一声的时候起过床。
- 战斗感觉（粗略）：节奏强制型——每回合固定时间点触发事件，若上一回合你“没有完成某种任务”，就给你疲劳/少抽牌。

4. 晚上舍友约你打游戏（RED/BLACK）
--------------------------------
- 场景：晚上躺床上 / 准备学习。
- 现实梗：“就一把”“再来最后一把”，结果写作计划全毁。
- 战斗感觉（粗略）：诱惑型——给你塞“爽牌”（高爆发/高抽），但打出会产大量心魔或下回合严重减能量。

5. 电脑网络炸了 / 电脑崩溃（BLUE/RED）
------------------------------------
- 场景：交稿前十分钟 / 正在跑实验。
- 现实梗：未保存的改动、没备份的数据，都在你按下 Alt+F4 的那一刻消失。
- 战斗感觉（粗略）：随机打断型——随机摧毁/锁定你的手牌/弃牌堆/抽牌，让你节奏瞬间崩溃。

6. 美女的小师妹（GREEN/BLUE）
---------------------------
- 场景：组会 / 实验室日常。
- 现实梗：人美、聪明、效率高，你只剩自卑和社恐。
- 战斗感觉（粗略）：对比打击型——自己行动不算凶，但每次出手都给你叠“自我怀疑”，降低你的输出或自信上限。

7. 投稿被审稿人几个月不审（BLUE/GREEN）
------------------------------------
- 场景：投稿系统界面 / 每日刷新状态。
- 现实梗：状态一直是“Reviewers invited”，你每天刷但什么都没有发生。
- 战斗感觉（粗略）：拖延蓄力型——前几回合几乎不动，只叠“拖延层数”，到一定层数后一次性给你心魔/大额压力。

8. 导师给你接横向找甲方（BLACK/WHITE）
------------------------------------
- 场景：导师办公室 / 临时通知。
- 现实梗：你本来想做自己的课题，却突然要去给甲方做报表和 PPT。
- 战斗感觉（粗略）：资源挪用型——每回合把你一部分能量/抽牌“转移”成没用的任务牌/诅咒牌，压缩你主牌组空间。

9. 甲方的折磨（BLACK/RED）
-----------------------
- 场景：项目汇报 / 甲方会议。
- 现实梗：项目已经做完一半，甲方说“我们想改个小地方”，结果全推倒重来。
- 战斗感觉（粗略）：反复重置型——频繁修改你的状态：清空 buff、重排手牌、重置你辛苦叠出的资源。

10. 论文写到底发现开头退错了（BLUE/BLACK）
---------------------------------------
- 场景：熬夜写论文的最后一晚。
- 现实梗：写了三万字，发现第一节立的前提不对。
- 战斗感觉（粗略）：后期毁灭型——当战斗/回合计数到一定程度时，突然“否定你之前的努力”：削减所有增益或强制弃掉关键牌。

11. 财务处的报账大妈（WHITE/RED）
-------------------------------
- 场景：行政楼财务处窗口。
- 现实梗：“发票没贴平”“签字不对”“系统关了”，哪怕你有一百个理由，她只有一个“不行”。
- 战斗感觉（粗略）：规则反伤型——设立苛刻的“出牌规则”（如只能打偶数牌/只能打攻击牌），一旦违反就强力反击或让牌无效。

12. 实验室半夜停电（BLUE/BLACK）
------------------------------
- 场景：通宵跑数据的实验室。
- 现实梗：不论是冰箱里的样品还是跑了三天的模型，都在那一秒的漆黑中化为乌有。
- 战斗感觉（粗略）：能量剥夺型——每隔几回合触发“断电”，直接清空你当前回合剩余能量，甚至扣除下回合能量。

13. 学术会议的社交晚宴（GREEN/WHITE）
-----------------------------------
- 场景：圆桌晚宴 / 自助餐区。
- 现实梗：虽然很想吃东西，但必须端着酒杯假装认识大佬，满脸堆笑其实脚趾扣地。
- 战斗感觉（粗略）：虚伪社交型——给你塞“客套话”状态牌（不能打出，占手牌位），你必须消耗能量去“应酬”（打出特定牌）才能消除。

14. 只有你会用的那台仪器坏了（BLUE/GREEN）
---------------------------------------
- 场景：排队等实验仪器。
- 现实梗：全组只有那台老旧的机器能做这个实验，然后它今天亮红灯了。维修师傅说下周才来。
- 战斗感觉（粗略）：关键封印型——随机“封印”你的一种颜色/类型的牌（如不能打出蓝色牌），持续数回合，逼你用不擅长的方式战斗。

15. 隔壁组发了和你一样的题目（BLACK/RED）
---------------------------------------
- 场景：刷 Arxiv / 朋友圈。
- 现实梗：你刚做出来的结果，别人昨晚发了预印本。你的创新点瞬间变成了“重复工作”。
- 战斗感觉（粗略）：竞速狂暴型——它会随着回合数增加而指数级变强（抢发了！），你必须在有限回合内（抢发前）击败它，否则直接判定失败或难度激增。

16. 审稿人意见打架（WHITE/BLACK）
-------------------------------
- 场景：收到 Reviewer Comments 的邮件。
- 现实梗：Reviewer 2 说“缺乏创新”，Reviewer 3 说“步子迈太大”。你改左边不是，改右边也不是。
- 战斗感觉（粗略）：双重束缚型——拥有两种状态（如“左倾”和“右倾”），每种状态下对特定牌免疫或反伤，你需要频繁切换自己的打法来适应。

17. 本科同学晒娃/晒年终奖（GREEN/BLACK）
--------------------------------------
- 场景：朋友圈 / 同学群。
- 现实梗：你还在为几百块的报销发愁，人家已经全款买车了。那一刻的心理落差比 Paper 被拒还大。
- 战斗感觉（粗略）：精神污染型——直接攻击你的“心魔”槽（如有），或者每回合塞入“嫉妒”诅咒牌（打出扣血，不打出占位），逼你面对心理阴暗面。

18. 突发的腰间盘突出（RED/GREEN）
-------------------------------
- 场景：久坐的实验室椅子 / 搬重物。
- 现实梗：突然的一阵剧痛，提醒你还没拿到学位，身体已经先垮了。
- 战斗感觉（粗略）：行动惩罚型——限制你每回合能打出的牌的数量（如“腰疼只能动3次”），或者每打一张牌都扣少量血（疼痛代价）。

19. 找不到原始数据的文件夹（BLUE/BLACK）
-------------------------------------
- 场景：写论文插图时 / 应对审查时。
- 现实梗：“New Folder (2)” 里是空的，“Final_v3” 里也是空的。你确信它存在，但就是找不到。
- 战斗感觉（粗略）：检索失效型——让你的抽牌堆/弃牌堆暂时“不可见”或“乱序”，或者让你原本能定向检索牌的能力失效，只能盲抽。

20. 毫无意义的表格填写（WHITE/BLUE）
-----------------------------------
- 场景：各类申报系统 / 年度考核。
- 现实梗：姓名、性别、出生年月...这些信息系统里明明都有，但你还是要一遍遍手动填，还不能复制粘贴。
- 战斗感觉（粗略）：机械重复型——要求你必须连续打出相同类型/费用的牌（填表），否则触发惩罚；强行打断你的流畅连招，逼你做无意义的操作。

21. 薛定谔的代码 Bug（BLUE/BLACK）
--------------------------------
- 场景：Demo 演示 / 组会汇报现场。
- 现实梗：自己在下面跑了一百遍都没事，导师一坐下来看，它就崩了。
- 战斗感觉（粗略）：概率失效型——你的卡牌在打出时有一定概率（如 20%）突然“失效”或产生反效果，模拟那种不可控的恐惧。

22. 永远养不活的细胞/植物（GREEN/RED）
------------------------------------
- 场景：生物培养箱 / 实验田。
- 现实梗：你像伺候祖宗一样伺候它们，它们还是说死就死，完全不讲道理。
- 战斗感觉（粗略）：培育反噬型——它会召唤出脆弱的“幼体”，你不能打死幼体（因为是珍贵的样品），但幼体会不断给你叠 Debuff，你必须小心翼翼地绕过它们攻击本体。

23. 让你帮忙取快递的师兄（RED/WHITE）
-----------------------------------
- 场景：去实验室的路上。
- 现实梗：“顺便帮我拿一下”“顺便帮我带个饭”，他的“顺便”是你的一整个中午。
- 战斗感觉（粗略）：负重缠绕型——每回合往你手牌里塞“快递”/“外卖”（高费，打出无效果，移除需消耗能量），占用你的手牌上限和费用。

24. 只有冷水的宿舍淋浴（GREEN/BLUE）
----------------------------------
- 场景：冬天的深夜宿舍。
- 现实梗：拖着疲惫的身体回去想洗个热水澡，结果喷头出来的全是冰水，那一刻心态崩了。
- 战斗感觉（粗略）：冰冻迟缓型——每回合“冻结”你的一张手牌（下回合才能用）或增加你卡牌的费用，模拟身体僵硬、行动迟缓的感觉。

25. 查重报告上的红色高亮（WHITE/BLACK）
-------------------------------------
- 场景：查重系统结果页。
- 现实梗：你自己写的句子被标红了，显示来源是“互联网资源”，相似度 100%。
- 战斗感觉（粗略）：自我否定型——当你打出高稀有度/高伤害的牌时，它会判定“查重不通过”（反制），迫使你只能用基础牌（“自己写的烂句子”）来慢慢磨血。

26. 参考文献格式爆炸（WHITE/BLUE）
--------------------------------
- 场景：EndNote / Zotero 软件界面。
- 现实梗：你只是删了一个引用，结果整个文档的 [1] 全变成了 [204]，格式全乱，修复需两小时。
- 战斗感觉（粗略）：格式错乱型——打乱你手牌的顺序或“费用”，让你原本规划好的连招（1->2->3）变得无法衔接（突然变成 1->5->2）。

27. 打扫卫生的阿姨（RED/GREEN）
-----------------------------
- 场景：实验室走廊 / 你的工位旁。
- 现实梗：阿姨为了吸地，拔掉了那个插着服务器/培养箱/恒温锅的插座。
- 战斗感觉（粗略）：无意秒杀型——平日里人畜无害（攻击力低），但每隔几回合有极小概率打出“拔电源”：直接结束你的回合，清空你所有已生效的持续性 Buff（因为断电了）。

28. 导师的“迷之微笑”（BLACK/WHITE）
----------------------------------
- 场景：导师办公室 / 走廊偶遇。
- 现实梗：你汇报完工作，他没说话，只是笑了一下。你开始疯狂脑补“他是不是觉得我是傻逼”。
- 战斗感觉（粗略）：猜疑心理型——它头上会挂一个“？”状态，你必须猜他下回合要干嘛（攻击/防御/Buff），猜错了你受重伤（心理防线崩塌），猜对了有奖励。

29. 咖啡因戒断反应（RED/BLUE）
----------------------------
- 场景：空荡荡的咖啡罐 / 下午三点。
- 现实梗：今天没喝咖啡，头痛欲裂，看文献像看天书，只想躺在地上。
- 战斗感觉（粗略）：成瘾依赖型——你每回合必须打出一张“提神”牌（或消耗特定资源），否则你会进入“戒断状态”：所有牌伤害减半，受到伤害加倍。

30. 不知何时过期的试剂（GREEN/BLACK）
----------------------------------
- 场景：试剂架 / 失败的实验结果。
- 现实梗：实验失败了一周，你才发现那瓶关键试剂半年前就过期了。
- 战斗感觉（粗略）：延迟隐患型——你在前几回合的操作看似正常生效，但其实埋下了“失败种子”；数回合后，之前的攻击/防御效果会被“回滚”或转化为对自身的伤害。
