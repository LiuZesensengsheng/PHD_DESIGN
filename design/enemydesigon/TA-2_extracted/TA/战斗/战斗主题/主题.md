# 战斗主题

助教线所有战斗都会有一个**主题**。目前设计有 **4** 个贴合助教日常的战斗主题，每个主题都有其特定的敌人组合与任务奖励/惩罚。  
此外设有**精英怪**：**无主题**（可插入任意战斗）、**强度固定**、**专属任务池**（仅从专属任务抽取，不用通用池）。  
→ 精英设计见 [DDL种植园主](../敌人设计/精英/DDL种植园主.md)、[微信私聊/QQ群聊/邮箱（三端）](../敌人设计/精英/三端（微信QQ邮箱）.md)，专属任务见 [任务列表 · 精英](../../任务系统/任务列表.md#精英)。

## 主题列表


| 主题                | 说明                                         |
| ----------------- | ------------------------------------------ |
| [答疑](答疑主题.md)     | 助教给本科生答疑，由好奇宝宝、助教搭子等组成                     |
| [点名](点名主题.md)     | 课前点名，需在迟到学生赶到前记缺勤                          |
| [交作业](交作业主题.md)   | 卷王往作业堆交作业，作业堆按 HP 比例造成伤害，需做目标取舍与节奏管理       |
| [小组考核](小组考核主题.md) | 逼王带普通学生小组考核，任务成功给自信、失败给小型 AOE；逼王从普通学生受伤中获益 |

## 敌人配置

| 强度等级  | [答疑](答疑主题.md)                                                 | [点名](点名主题.md)                                                                           | [交作业](交作业主题.md)                                              | 小组考核                                                          |
| ----- | ------------------------------------------------------------- | --------------------------------------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------- |
| **1** | [好奇宝宝](../敌人设计/答疑/好奇宝宝.md) x2                                 | [迟到学生](../敌人设计/点名/迟到学生.md) x1、[代到学生](../敌人设计/点名/代到学生.md) x1、[拖延怪](../敌人设计/点名/拖延怪.md) x1 | [卷王](../敌人设计/交作业/卷王.md) x2（固定召唤唯一 [作业堆](../敌人设计/交作业/作业堆.md)） | [逼王](../敌人设计/小组考核/逼王.md) x1 带 [普通学生](../敌人设计/小组考核/普通学生.md) x1 |
| **2** | [好奇宝宝](../敌人设计/答疑/好奇宝宝.md) x3                                 | [迟到学生](../敌人设计/点名/迟到学生.md) x2、[代到学生](../敌人设计/点名/代到学生.md) x1、[拖延怪](../敌人设计/点名/拖延怪.md) x1 | [卷王](../敌人设计/交作业/卷王.md) x3（固定召唤唯一 [作业堆](../敌人设计/交作业/作业堆.md)） | [逼王](../敌人设计/小组考核/逼王.md) x1 带 [普通学生](../敌人设计/小组考核/普通学生.md) x2 |
| **3** | [好奇宝宝](../敌人设计/答疑/好奇宝宝.md) x3 + [助教搭子](../敌人设计/答疑/助教搭子.md) x1 | [迟到学生](../敌人设计/点名/迟到学生.md) x3、[代到学生](../敌人设计/点名/代到学生.md) x1、[拖延怪](../敌人设计/点名/拖延怪.md) x1 | [卷王](../敌人设计/交作业/卷王.md) x4（固定召唤唯一 [作业堆](../敌人设计/交作业/作业堆.md)） | [逼王](../敌人设计/小组考核/逼王.md) x1 带 [普通学生](../敌人设计/小组考核/普通学生.md) x3 |


*说明：点名主题中「代到学生」为固定配置；交作业主题以 [卷王](../敌人设计/交作业/卷王.md) 为核心并召唤 [作业堆](../敌人设计/交作业/作业堆.md)；小组考核为 [逼王](../敌人设计/小组考核/逼王.md) 带 [普通学生](../敌人设计/小组考核/普通学生.md)。*

## 主题任务奖惩一览


| 主题                | 任务成功奖励         | 任务失败惩罚          |
| ----------------- | -------------- | --------------- |
| [答疑](答疑主题.md)     | 获得 1 自信        | 失去 1 自信         |
| [点名](点名主题.md)     | 增加迟到学生到达教室的倒计时 | 减少迟到学生到达教室的倒计时  |
| [交作业](交作业主题.md)   | 减少【作业堆】10% HP  | 增加【作业堆】5% HP 上限 |
| [小组考核](小组考核主题.md) | 获得 1 自信        | 对场上所有目标造成 5 点伤害 |
