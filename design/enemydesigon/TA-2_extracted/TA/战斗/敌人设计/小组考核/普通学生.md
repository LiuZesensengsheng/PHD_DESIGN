# 普通学生

> 所属主题：[小组考核主题](../../战斗主题/小组考核主题.md) · 与 [逼王](逼王.md) 同场出现，作为其「小组」成员

---

## 基本信息（侧写）

| 项目         | 内容 |
|:-------------|:-----|
| **名字**     | 普通学生 |
| **敌人类别** | 普通 |
| **出现时机** | 小组考核主题中，由配置决定数量（x1～x3） |
| **关键词**   | 跟组、划水、背景板 |
| **形象**     | 大众脸路人感：普通卫衣/格子衫、低头看手机或发呆，存在感弱；站在逼王旁边时更像「被带的那一个」。 |
| **背景**     | 大课里最常见的那种：不卷也不摆，作业按时交、考试能过就行。被分到和逼王一组时内心 OS 是「别 cue 我」，但架不住逼王非要代表全组发言、抢答、质疑助教，自己只好在旁边点头或沉默。 |

---

## 机制（白板）

- **无被动**
- **行动循环**：仅**基础普攻**（如：造成 3 点伤害），无特殊技能。  
  *设计上刻意保持白板，让战场焦点在 [逼王](逼王.md)；普通学生主要提供「数量」和「被逼王利用」的靶子。*

---

## 台词（风味化）

1. 「……我都行，你们定吧。」
2. 「呃，这道题我还没看。」
3. 「没事没事，你说你说。」
4. 「（小声）能不能别老代表我们组……」
5. 「反正有他在，我们组应该能过吧。」

---

*[逼王](逼王.md) 的机制会围绕【普通学生】受伤时获得治疗/力量；清掉普通学生可削弱逼王的续航与叠层。*
