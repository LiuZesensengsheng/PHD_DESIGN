# 遭遇/敌人灵感 Backlog（随手记）

> 目的：把“灵光一现”的点子先记下来，避免丢；后续再挑优先级写成完整 encounter 文档。
> 书写原则：每个点子先用 1 句话说**情绪与画面**，再用 1–2 条说**机制轴**（不要一上来堆复杂度）。

## 数学主题（微积分 / 变分等）
- **微积分敌人组**：求导 / 微分 / 积分 / 变分，把玩家的行为“数学化”为代价或收益。
  - **机制轴候选**：延迟结算、累积阈值、约束规则、违反惩罚（λ/利息）、“越做越痛”的反噬。

## 计算机主题（循环 / 递归 / ……）
- **循环（Loop）**：重复劳动、无尽 debug、无意义迭代。
  - **机制轴候选**：回合循环/意图重复、复读机式 Debuff、周期性增幅（每 N 回合一次爆发）。
- **递归（Recursion）**：问题套问题，越解决越深。
  - **机制轴候选**：生成“子问题/小怪”、分裂/复制、把玩家的牌“包一层”再返还（嵌套效果）。

## 化学主题
- **化学实验室敌人组**：反应、催化、沉淀、滴定、爆沸、通风橱与安全培训。
  - **机制轴候选**：配比（资源颜色/能量比例）、反应条件（满足条件触发强效果）、副反应（随机负面）、催化（加速计时/加倍效果）。

## 生物主题（过柱子 / 养细胞）
- **过柱子（柱层析）**：跑条、分离、洗脱，盯着色带一点点爬。
  - **机制轴候选**：进度条/洗脱条、分段阈值奖励或惩罚、选择洗脱策略（快但损失/慢但安全）。
- **养细胞**：污染、传代、换液、冻存，永远不确定明天还活不活。
  - **机制轴候选**：脆弱单位（容易“污染”死亡但可带来增益）、维护成本（每回合必须投入资源/行动）、随机事故事件。

### 细胞/病原体/免疫组合素材（偏恐怖也偏燃）
> 关键词池：共培养 / 感染 / 潜伏 / 裂解 / 污染 / 耐药 / 炎症 / 清除 / 免疫点刹车

- **永生化细胞系（HeLa / “永生细胞”）**：长得太快，抢资源，还会“污染”你的体系。
  - **机制轴候选**：指数增殖/召唤分裂体；“占位”挤压玩家资源；击杀后留下残留污染（持续 Debuff）。
- **支原体（Mycoplasma）污染**：看不见，但你感觉一切都慢慢不对劲。
  - **机制轴候选**：隐形 Debuff（抽牌/能量/治疗效率下降）；被“检测”前不可选中/不可完全移除。
- **衣原体（Chlamydia）**：细胞内寄生，前期很安静，后期突然爆发。
  - **机制轴候选**：潜伏计数器（若干回合后触发爆发）；“夺取宿主”（附着到某单位/某张牌上改变其行为）。
- **噬菌体（Phage）**：专门猎杀细菌，但裂解释放的东西会让场面更乱。
  - **机制轴候选**：对“细菌”标签目标有特攻并触发裂解爆发；裂解产生碎片/毒素/混乱（副作用）。
- **生物膜（Biofilm）**：不是更强，是更难被处理掉。
  - **机制轴候选**：随时间增长的减伤/护盾；需要满足“拆膜条件”（特定顺序/特定类型）才能正常伤害。
- **质粒（Plasmid）传播**：能力像传染一样在敌人之间流动。
  - **机制轴候选**：转移特性（耐药/增殖/反伤）；优先传给最危险或血量最低者。
- **抗生素筛选（Selection Pressure）**：你以为在清场，其实在培养耐药。
  - **机制轴候选**：重复使用同类手段会提升“耐药层数”并降低效果；达到阈值触发“旁路激活”（换形态/换规则）。
- **巨噬细胞（Macrophage）**：清理垃圾、吞噬小怪，但炎症会失控。
  - **机制轴候选**：吞噬（移除小单位/状态）+ 炎症层数（过高反噬玩家/全场）。
- **T 细胞（T Cell）点名**：识别“抗原”并持续追击。
  - **机制轴候选**：抗原标记（某目标/某行为/某类牌）→ 被标记者持续受压；玩家可通过“伪装/免疫逃逸”解除。
- **NK 细胞（Natural Killer）**：专盯“异常表达”的目标。
  - **机制轴候选**：某类 Buff/状态越高越容易被锁定清除（反制堆叠流）。
- **细胞因子风暴（Cytokine Storm）**：全场加速与增伤，但回合末统一衰竭。
  - **机制轴候选**：短期强增益（加速/增伤）+ 回合末反噬（失去生命/失去资源/抽牌下降）。

- **培养箱噩梦（恐怖向）**：永生细胞 + 支原体污染 + 衣原体潜伏

  - **机制轴**：清除/点名 + 加速但反噬

## 概率/统计/预测主题（采样 → 统计 → 预测 / 熵）
- **随机性流水线**：你以为在“分析数据”，其实是在被数据反过来分析。
  - **机制轴候选**：不确定性 token（方差/噪声）、抽样偏差（信息缺失/被遮蔽）、过拟合（短期强/长期崩）、熵（越拖越混乱或越难预测）。

## 数学符号/常数主题（i / e / π）
- **符号崇拜与不可理解**：你不是在“算”，你在被符号统治。
  - **机制轴候选**：相位/虚实切换（i）、指数增殖/复利（e）、周期/重复/回归（π）、常数不可改变（锁定规则）。

## 机器学习主题（Embedding / Transformer / LSTM / …）
- **Embedding（嵌入）**：把一切“压扁成向量”，然后用相似度决定你的命运。
  - **机制轴候选**：相似度标记（某类牌越像越被惩罚/越被奖励）、聚类（给牌/敌人打簇标签）、“最近邻”锁定（你重复动作会被点名）。
- **Transformer（注意力）**：不是你在选择目标，是“注意力”在选择你该注意什么。
  - **机制轴候选**：Attention 指向（每回合标记 1–2 个目标/牌位，触发额外效果）、Softmax 竞争（被标记者承受更高伤害/更高反击）、多头（多个小规则同时生效但各自很简单）。
- **LSTM（长短期记忆）**：它记得你做过的每一件小事，并在关键时刻翻旧账。
  - **机制轴候选**：短期缓存（本回合行为立即惩罚）+ 长期记忆（累计到阈值爆发）、遗忘门（玩家可用某种方式“清空记忆”换取代价）。

## 生物医学主题（靶向疗法 / 免疫治疗 / …）
- **靶向疗法（Targeted Therapy）**：你以为你在精确打击，其实对方在“耐药进化”。
  - **机制轴候选**：靶点标记（你每次打同一“靶点”会更有效）+ 耐药层数（重复打会逐渐失效并反噬）、旁路激活（耐药后换一种更难缠的形态）。
- **免疫治疗（Immune Checkpoint）**：你不是缺伤害，你被“刹车”锁死了。
  - **机制轴候选**：检查点锁（限制某类行动）与解锁窗口（满足条件解除并获得爆发回报）。

## 其他学科主题（先建库，后细化）
- **经济学（激励相容）**：每个选择都在被“机制设计”定价。
  - **机制轴候选**：奖励/惩罚对称、逆向激励（你越想拿某收益越被收税）。
- **博弈论（纳什均衡）**：你最优，对面也最优，结果大家一起难受。
  - **机制轴候选**：同步出招/反制、读心式惩罚（你偏离“均衡”就吃亏）。
- **控制论（反馈回路）**：你每次修正都会带来过冲与震荡。
  - **机制轴候选**：正反馈（越做越强/越糟）与负反馈（抑制但代价）、震荡（状态来回切）。
- **信号处理（滤波/噪声）**：你看到的是“滤波后”的现实。
  - **机制轴候选**：信息遮蔽、噪声层数、关键回合“去噪”窗口。
- **密码学（哈希/签名/密钥）**：你必须证明你“有资格”才能行动。
  - **机制轴候选**：钥匙 token、验证失败惩罚、一次性密钥（用掉就没）。
- **编译原理（语法树/优化）**：你打出的牌会被“重写/优化/删掉死代码”。
  - **机制轴候选**：重写规则（某些连招被改写）、死代码（某类牌暂时无效）、优化窗口（满足条件奖励更强效果）。
- **数据库（事务/锁/死锁）**：你不是输了，你被锁住了。
  - **机制轴候选**：锁定资源、死锁（两边都卡住但你更亏）、提交/回滚（延迟结算）。
- **操作系统（调度/抢占）**：轮到你了吗？不一定。
  - **机制轴候选**：行动顺序被抢占、优先级、上下文切换（状态重置/错位）。
- **网络（拥塞控制）**：越急越堵，越堵越掉包。
  - **机制轴候选**：每回合“拥塞”层数、爆发会触发掉包（弃牌/费用上升）。
- **数值分析（迭代收敛）**：你在逼近答案，但永远差一点。
  - **机制轴候选**：误差项（越小越难降）、迭代次数阈值（超过就发散/惩罚）。
- **材料/化工（工艺窗口）**：条件稍微偏一点，产率就崩。
  - **机制轴候选**：窗口判定（满足条件强奖励，否则副反应）、产率/纯度两条进度条。
- **土木/工地（打灰/放线/钢筋/验收）**：你不是在战斗，你在赶工期、过验收、还要不出事故。
  - **机制轴候选**：工序/流程（必须按顺序做事）、工期 DDL（越拖越罚）、材料配比（资源投入不对就返工）、安全红线（触发就重罚）。
- **心理学（认知偏差）**：你以为你在决策，其实是偏差在决策。
  - **机制轴候选**：损失厌恶（失去资源更痛）、确认偏误（重复选项越上头越亏）。
- **法学/伦理（合规审查）**：你不是不能做，是“暂时不合规”。
  - **机制轴候选**：合规条、审查回合、违规惩罚与合规奖励。

### 土木/工地素材（更“实践”，更有声音和粉尘）
- **搅拌车（Concrete Mixer）**：材料在转，你的时间在掉。
  - **机制轴候选**：倒计时（到点必须“浇筑/打出某类牌”否则报废）；投入资源越多，下一回合收益越大（但也更容易“离析/爆裂”）。
- **泵车/浇筑（Pump & Pour）**：一旦开泵就不能停。
  - **机制轴候选**：连段要求（连续回合必须投入行动）；中断会触发“冷缝/返工”惩罚。
- **钢筋笼（Rebar Cage）**：硬，密，拆不掉。
  - **机制轴候选**：高护甲/减伤；需要先“绑扎/切断”若干次才能正常伤害（条件破防）。
- **模板（Formwork）**：限制形状，限制自由。
  - **机制轴候选**：限制规则（本回合某类牌不可用/费用提高）；满足“拆模条件”后解除并获得奖励窗口。
- **放线/测量（Surveying）**：你得先对准，才能继续施工。
  - **机制轴候选**：命中/偏差条；偏差过大则“返工”（弃牌/回退进度）。
- **养护（Curing）**：看起来没做事，其实最关键。
  - **机制轴候选**：延迟收益（现在投入换未来护盾/回血）；若被打断会触发裂缝/强度不足（永久负面）。
- **质量/安全检查（QA/QC & Safety）**：你可以很快，但你敢吗？
  - **机制轴候选**：安全红线计数器（违规叠层，达到阈值触发事故大罚）；“抽检回合”若不合规直接重置部分进度。
- **工期/赶工（Schedule Crunch）**：不是打不过，是来不及。
  - **机制轴候选**：工期 DDL（回合数阈值后敌人强化/玩家受罚）；提前完成会给额外奖励（绩效/奖金）。
- **塔吊/吊装（Tower Crane & Lifting）**：你以为是“搬运”，其实是高空风险与节拍控制。
  - **机制轴候选**：节拍回合（某些回合“吊装窗口”收益更高）；失误/违规叠加“事故风险”并触发大罚。
- **脚手架（Scaffolding）**：临时结构也会变成永久麻烦。
  - **机制轴候选**：临时护盾/地形改变（影响站位/目标选择）；超期不拆会转化为负面状态（占位/费用上涨）。
- **极端天气停工（Heatwave / Storm Stop-work）**：不是你不努力，是天气不给你干。
  - **机制轴候选**：随机或可预告的“停工回合”（行动受限/收益打折）；提前做“防护/预案”可减轻惩罚。
- **监理签字（Supervisor Sign-off）**：你可以做完，但没有签字就等于没做。
  - **机制轴候选**：关键节点需要“验收动作/材料证明”才能推进；未通过则回退/返工（延迟结算的负面）。
- **甲方变更（Client Change Order）**：规则突然改了，但锅还是你的。
  - **机制轴候选**：随机更改战斗规则/目标（某类牌暂时失效/费用上升/优先级变动）；应对成功会获得“追加费用/补偿”奖励，失败则叠“返工债务”。

#### 现成组队配方
