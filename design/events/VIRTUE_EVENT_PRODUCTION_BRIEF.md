# 生产简报：【初心的考验】事件 v1.2 (混合版)

**文档目标**: 本简报定义了一个兼具《暗黑地牢》式冲击力与核心选择的审判事件。它采用纯UI驱动，但引入了关键的“折磨后抉择”机制。

**核心原则**: **暂停战斗，强制关注，后果抉择**。

---

## 统一流程 (Single-Act Flow with Choice)

| 步骤 | 资产需求 (Art & Sound) | 程序实现 (Programming) |
| :--- | :--- | :--- |
| **1. 触发** | **UI**: 风格化的“**初心的考验**”文字横幅。 | **Trigger**: 玩家压力首次`>= 100`，触发`RESOLVE_TEST_START`事件。 |
| | **SFX**: （本版不使用） | **Flow**: **暂停战斗** (`game.pause()`)，玩家暂时无法操作。 |
| | | **UI**: 在角色上方显示横幅，1.5秒后淡出。 |
| **2. 结果揭示** | **UI**: 为每个美德/折磨状态制作一个**带背景的半屏立绘**，上面有巨大、风格化的结果文字（如**【掌控】**）。 | **Logic**: 调用`JudgmentService.decide_verdict()`，获得结果（美德或折磨）。 |
| | **VFX**: **美德**: 角色迸发温暖闪光。**折磨**: 角色溅射黑暗能量。 | **UI**: 弹出对应的结果立绘和文字，持续2-3秒。 |
| | **字幕**: 角色发表对应的**独白文本**（例如：“秩序，在此宣告！”或“一切……都失控了……”）。 | **字幕**: 显示独白字幕3秒（无语音文件）。 |
| | **仅美德：信念抽卡** | **Card**: `draw(1)`；将抽到的卡牌高亮1秒（手牌内）。 |
| | **仅美德：理想降临（最强姿态）**：若抽到的卡牌与触发本次审判的理想为同色，且为【课题卡牌】（或放宽为同色卡），则显示小横幅“理想降临”。 | **Logic**: 为该卡牌添加临时效果：本回合费用=0；其“主效果”在结算完成后再结算1次（仅主句，不复制衍生/跨回合）。全战至多触发1次；与“信念余晖”不叠加。 |
| **3. 后续处理** | **UI**: 为每个状态制作一个小的HUD图标。 | **IF Result == VIRTUE (美德)**: |
| | **UI (折磨专属)**: 一个带有“是/否”选项的对话框，文本为“是否永久要放弃[理想名]，彻底摆脱此折磨？”。 | > 调用`apply_verdict(VIRTUE)`，为玩家添加被动状态。 |
| | | > 在HUD添加状态图标。 |
| | | > **战斗恢复** (`game.resume()`)。 |
| | | **ELSE IF Result == AFFLICTION (折磨)**: |
| | | > 弹出“是否放弃理想”的**模态对话框**。 |
| | | > **IF Player_chooses_YES**: |
| | | > > 调用`abandon_ideal()`，移除理想，大幅降低压力，**不添加**折磨状态。 |
| | | > > **战斗恢复**。 |
| | | > **ELSE IF Player_chooses_NO**: |
| | | > > 调用`apply_verdict(AFFLICTION)`，为玩家添加折磨状态。 |
| | | > > 在HUD添加状态图标。 |
| | | > > **战斗恢复**。 |

## 附注：关于“裂痕”与“粉碎”

-   **裂痕**: 已被新的“折磨后抉择”流程所吸收和替代，不再作为一个独立的审判结果。
-   **粉碎**: 可以在`decide_verdict()`中作为一个更高压力的检定结果（例如压力 > 150），直接施加一个不可移除的超级折磨，且**不给**玩家放弃理想的机会。这是V2可以考虑的扩展。
