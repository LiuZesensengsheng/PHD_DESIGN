# 发明专利请求书信息模板

版本：v0.1

说明：本文件不是正式请求书，而是填写国家知识产权局专利业务办理系统前的信息整理模板。不要在公开仓库中填写身份证号、手机号、详细住址等敏感信息。

## 一、发明名称

建议名称：

一种基于压缩语义空间的卡牌游戏内容设计评估方法、系统、设备及存储介质

备选名称：

一种基于压缩语义表示的组合式数字内容对象设计评估方法、系统、设备及存储介质

选择建议：

- 若希望投资人和团队内部理解更直接，使用第一个名称。
- 若担心“卡牌游戏”被审查员理解为偏游戏规则，可以和正式文本一起考虑第二个更上位名称。

- 多申请人时，需要提前约定权利比例和后续转让规则。

## 六、费用减缴

## 七、文件清单

准备上传或录入的文件：

- 发明专利请求书：在系统中填写。
- 权利要求书：由 `02_claims_draft.md` 整理为正式格式。
- 说明书：由 `03_specification_draft.md` 整理为正式格式。
- 说明书摘要：由 `04_abstract_and_drawings_draft.md` 整理。
- 说明书附图：根据 `04_abstract_and_drawings_draft.md` 绘制正式黑白图。
- 摘要附图：建议选图1。

内部留存文件：

- `01_invention_disclosure.md`
- benchmark 证据和测试结果
- 发明人贡献记录
- 提交回执和申请号

## 八、提交前最终确认

- 已确认申请类型为发明专利。
- 已确认申请人和发明人。
- 已确认没有在申请前公开核心技术细节。
- 已确认权利要求书和说明书术语一致。
- 已确认摘要不超过 300 字。
- 已确认附图没有彩色、照片、产品 UI 或商业秘密。
- 已保存所有提交文件的最终版本。
