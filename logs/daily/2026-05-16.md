  contract, but it must not change `QuestLoader.load_all()` or runtime JSON
  loading behavior.
- `slack` remains an allowed empty `event_source` pack with no runtime outputs.

  a loader, resolver activation layer, save-pinning source, or hot-reload
  surface.

  or change `QuestLoader`.

  still does not call `QuestLoader.load_all()`, load runtime JSON, or make
  runtime paths authoritative.
